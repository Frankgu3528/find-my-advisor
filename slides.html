How It Works

Three steps.
From a blank page to a shortlist.

01
Upload your materials
Drop in your CV and draft SOP. The system parses them and builds a structured academic profile — research interests, technical background, prior work — automatically.
02
Agent researches in real time
An autonomous agent browses faculty homepages, grant databases, and recent publications — gathering live signals that static profiles never surface.
03
Receive your personalized shortlist
A ranked list of advisors with match scores, verified tags, and targeted SOP revision suggestions — specific to each advisor's current research direction.

Semantic Recall.

Stage 03 · Personalized Output

Stage 03:
Personalized Output.

OUTPUT 01 · ADVISOR CARDS
Research tags
3–4 keyword labels extracted from live web content — reflects what the professor is working on now
Recruiting status
Yes / No / Unknown — inferred by the agent from current homepage and lab news
Key finding
One specific discovery — a recent paper, grant, or open position found during research
Match rationale
2–3 sentences explaining why this advisor fits the student's background and interests
OUTPUT 02 · SOP REVISION
After selecting an advisor, the user can request targeted revision suggestions for their SOP. Each suggestion contains:
Original
A passage from the user's uploaded CV or SOP that could be strengthened
Revised
A rewritten version that echoes the advisor's research language and current projects
Reason
One sentence explaining why the revision improves alignment with this specific advisor

LLM Integration

Language models are used
at five points in the pipeline.

CV / SOP Parsing LLM
User's uploaded materials are parsed and combined into a query that feeds the embedding retrieval step
ReAct Agent Loop ReAct
Drives the full Thought → Action → Observation loop — decides what to search, reads results, updates understanding, loops up to 4 times per professor
Report Generation ReAct
At loop end, LLM outputs a structured JSON — tags, recruiting status, match rationale, key finding
Ranking ReAct
The score in each report is LLM-generated — candidates are sorted by this score to produce the final top-10
SOP Revision LLM
Generates paragraph-level suggestions grounded in the advisor's research — original → revised → reason
